# Existential Crisis - Mod do Terraria

Um mod brasileiro que tem como objetivo criar um NPC cômico que tem crise existencial.

Sim, é uma referência ao Butter do Rick and Morty.

## Como instalar?

Atualmente, não estamos no Workshop e talvez até devido a políticas da Steam não é legal publicarmos.

Esse projeto é open source, mas tome cuidado com Forks, pois esse mod injeta DLLs e talvez essas DLLs estejam infectadas nos clones desse projeto. Use por sua conta em risco!

Então lido, o passo-a-passo é:

1. Realize um clone na sua máquina.
2. Execute a solução do projeto.
3. No menu do projeto (via workshop), coloque o seu GEMINI_API_KEY. Para saber mais sobre Api Key do Gemini, use o AI Studio do Google.
4. Divirta-se!

## Como contribuir?

1. Não aceitamos Pull-Requests que alterem as DLLs. Apenas o dono do repositório (@vinihiga) poderá fazer isso.
2. Apenas altere o código fonte em .net C#.
3. Seja sucinto no Pull-Request explicando tudo. Pode user vibe code, mas saiba o que está sendo feito.
4. Utilize padrões de projeto (design patterns). Boas práticas de Clean Code. Aplique SOLID sempre que possível.
5. Constantes deverão ser do SEGUINTE_FORMATO. Já variáveis em camelCase. Métodos, Classes e NameSpaces sempre PascalCase.
